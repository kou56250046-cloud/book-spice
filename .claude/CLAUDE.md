# プロジェクト設定：人生のスパイス

> 人生にひと匙の刺激を。

## プロジェクト概要

日本語のベストセラー本をジャンル別にすぐ探せる個人用読書ガイドサイト。
楽天ブックスAPIでデータを毎週土曜に自動取得し、GitHub Pages で静的公開する。

## 技術スタック

| レイヤー | 採用技術 |
|----------|----------|
| フロントエンド | HTML / CSS / Vanilla JavaScript（ライブラリ最小限） |
| フォント | Shippori Mincho B1（明朝・見出し）/ Noto Sans JP（本文） |
| データ形式 | JSON（books.json・comments.json） |
| ホスティング | GitHub Pages |
| 自動化 | GitHub Actions（cron: 毎週土曜 22:00 UTC = 07:00 JST） |
| データソース | 楽天ブックス書籍検索API / ランキングAPI（無料） |
| ユーザーデータ | localStorage（端末内のみ、サーバー不要） |

**重要：** Next.js・Supabase・Stripe・Vercel・認証機能は一切使用しない。完全無料・静的サイト。

## リポジトリ構成

```
spice-books/
├── index.html                 ← メインページ（唯一のHTMLファイル）
├── books.json                 ← 書籍データ（Actions が毎週自動生成）
├── comments.json              ← コメント（オーナーが手書き管理）
├── scripts/
│   └── fetch-books.js         ← 楽天API取得スクリプト（Node.js）
└── .github/
    └── workflows/
        └── weekly-update.yml  ← 毎週土曜 07:00 JST 起動
```

## 機能一覧

### コア機能（F-01〜F-06）
- F-01: ジャンル別ベストセラー表示（各20冊）
- F-02: スパイス棚タブでジャンル切り替え
- F-03: 書籍カード（書影・タイトル・著者・スパイス度・コメント）
- F-04: 楽天ブックスへの購入リンク
- F-05: 毎週土曜 07:00 JST 自動更新
- F-06: 今週のひと匙（トップ特集）

### 追加機能（F-07〜F-11）
- F-07: スパイス度（菱形の種粒メーター、1〜5段階）
- F-08: お気に入り・読了チェック（localStorage）
- F-09: キーワード検索・絞り込み（クライアント側）
- F-10: 一言レビュー・名言メモ（localStorage）
- F-11: 並べ替え（スパイス度順・ランキング順・タイトル順）

## データ設計

### books.json（API自動生成）
```json
{
  "isbn": "9784...",
  "title": "書籍タイトル",
  "author": "著者名",
  "publisher": "出版社",
  "genre": "self",
  "imageUrl": "https://...",
  "price": 1650,
  "reviewAverage": 4.5,
  "reviewCount": 312,
  "itemUrl": "https://books.rakuten.co.jp/...",
  "salesRank": 1,
  "spiceScore": 4.2,
  "fetchedAt": "2026-06-14"
}
```

### comments.json（オーナー手書き）
```json
{
  "9784...": {
    "reason": "おすすめの理由（1〜2文）",
    "view": "人生にスパイスを与える視点（1文）"
  }
}
```

### スパイス度の算出式
```
スパイス度（1〜5）=
  レビュー平均評価（5点満点）× 0.5
  + レビュー件数（対数スケールで正規化）× 0.3
  + ランキング上位補正 × 0.2
```

## デザイン要件

- 背景：和紙色 `#ECE3D0` ＋ 微細なドットパターン
- 見出し：Shippori Mincho B1（明朝体）
- スパイス度：★ではなく**菱形の種粒メーター**（サイト固有の署名要素）
- ジャンルは各スパイス色で色分け
- 余白を広めにとった書籍カードレイアウト
- モバイルファースト・レスポンシブ対応

## 画面構成

| エリア | 内容 |
|--------|------|
| ヘッダー | サイト名・更新日・検索バー・並べ替え・絞り込みボタン・ジャンル棚タブ |
| 今週のひと匙 | トップに今週イチ押しの本を大きく特集表示 |
| 書籍グリッド | 選択ジャンルの上位20冊をカード形式で表示 |
| 書籍カード | 書影・タイトル・著者・スパイス度・コメント・お気に入り・読了・メモ欄 |
| フッター | 楽天クレジット表記・localstorage の注意書き |

## 秘密情報の管理

- 楽天アプリIDは **GitHub Actions Secrets** に格納（`RAKUTEN_APP_ID`）
- コードに直接埋め込み禁止
- アフィリエイトは不使用

## コンプライアンス

- 楽天ウェブサービス利用規約を遵守
- フッターに楽天クレジット表記を必ず掲載（規約必須）
